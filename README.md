## Deployment
1. Clone the repository.
2. Run a local static server (recommended; ES modules often break on `file://`):
   - `python -m http.server 8080`
   - then open `http://localhost:8080`
3. Optional: connect real external feeds by replacing `external_data_fetcher_stub.js`.

## Symbolic Entanglement
Agents inherit symbolic fragments based on:
- internal recursive events (imprints, epoch collapse, drift signals)
- external real-world data (economic stress, cultural meme shifts, anomaly feeds)

This simulates agent evolution as a reflection of a collective symbolic field.
