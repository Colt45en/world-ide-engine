# Fresh World Engine

A custom game engine for creating interactive worlds with AI integration.

## Project Structure

- `ai/`: AI-related modules and scripts
- `assets/`: Game assets (images, sounds, models)
- `config/`: Configuration files
- `engine/`: Core engine components
- `game/`: Game-specific logic
- `graphics/`: Graphics rendering modules
- `knowledge/`: Knowledge base or data
- `math/`: Mathematical utilities
- `scripts/`: Build and utility scripts
- `studio/`: Editor or development tools
- `tests/`: Unit and integration tests

## Setup

1. Ensure Python 3.13+ is installed.
2. Create a virtual environment: `python -m venv .venv`
3. Activate the environment: `.venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`

## Development

Run the engine with: `python main.py`
